# 🖼️ 图片右下角马赛克工具

一个简单易用的网页工具，为图片右下角批量添加马赛克效果。

## ✨ 功能特点

- 📁 **批量处理** - 一次上传多张图片
- 🎛️ **灵活调节** - 可调整马赛克大小、位置和强度
- 👁️ **实时预览** - 处理前查看效果
- 📦 **一键下载** - 自动打包为 ZIP 文件
- 🔒 **本地处理** - 图片不上传服务器，安全隐私

## 🚀 使用方式

### 方式一：GitHub Pages（推荐）
直接访问：`https://你的用户名.github.io/image-mosaic-tool/`

### 方式二：本地使用
1. 下载 `index.html` 文件
2. 用浏览器打开即可使用

### 方式三：本地服务器
```bash
# Python 3
python3 -m http.server 8000

# 或 Node.js
npx serve
```
然后访问 `http://localhost:8000`

## 🎛️ 参数说明

| 参数 | 说明 | 范围 |
|------|------|------|
| 马赛克宽度 | 马赛克区域宽度占图片的百分比 | 5-50% |
| 马赛克高度 | 马赛克区域高度占图片的百分比 | 5-50% |
| 距右边距离 | 马赛克距离图片右边的距离 | 0-30% |
| 距底边距离 | 马赛克距离图片底边的距离 | 0-30% |
| 马赛克强度 | 马赛克像素块大小 | 5-50 |

## 📝 技术栈

- 纯 HTML/CSS/JavaScript
- JSZip - 用于打包下载
- 本地 Canvas 处理 - 无需后端
